ThreeJS-画点

发表于 2019-07-09 | 分类于 前端
如何使用ThreeJS来绘制点

画点

Three.js 中，与点相关的类API主要是Points(创建点的类), PointsMaterial(点相关的材质), BufferGeometry(基础几何体)

思路

  1. 定义基础几何体。可以参考文档中示例代码，three中每个顶点具有三维属性(x, y, z)。故顶点是最少三个值的数组队列。 通过 addAttribute 把值赋予 position
  2. 定义材质。参考文档对材质的色彩，透明度，纹理等进行相关参数设置
  3. 以几何体和材质做为参数创建点的类
  4. 把类添加到场景中

相关API

几何体(BufferGeometry)

链接

点材质(PointsMaterial)

绘制点所用的默认材质

示例

//This will add a starfield to the background of a scene
var starsGeometry = new THREE.Geometry();

for ( var i = 0; i < 10000; i ++ ) {

  var star = new THREE.Vector3();
  star.x = THREE.Math.randFloatSpread( 2000 );
  star.y = THREE.Math.randFloatSpread( 2000 );
  star.z = THREE.Math.randFloatSpread( 2000 );

  starsGeometry.vertices.push( star );

}

var starsMaterial = new THREE.PointsMaterial( { color: 0x888888 } );

var starField = new THREE.Points( starsGeometry, starsMaterial );

scene.add( starField );

参数：
PointsMaterial( parameters : Object )

参数是一个Object，key-value 都是基于基类Material，可参考Material文档
唯一例外的是颜色属性 color, 它使用 16进制字符串传入，默认为 0xffffff。

属性：

公用属性可以参考 基类Material

Properties values marks
.color Color
default: 0xffffff		 定义材质的颜色（16进制字符串）
.isPointsMaterial Boolean
default: true		 用于检查该类或者派生类是否为一个点材质。此类不应该被更改。
.map Texture 对材质使用指定纹理
.morphTargets Boolean
default: false		 定义材质是否使用 morphTargets （注）
.size Number
default: 1.0		 设置点的大小
.sizeAttenuation Boolean
default: true		 点的大小是否根据摄影机的视觉深度变化。(如果设为false则无论摄影机透视距离多少，点的大小都一样)

点类(Points)

用于呈现点的类

参数：

parameters values marks
geometry (可选)Geometry 几何体实例
material (可选)Material 材质

属性：

公用属性可以参考 基类Ojbect3D

Properties values marks
.geometry Geometry 定义几何体
.isPoints Boolean
default: true		 用于检查该类或者派生类是否为一个点。此类不应该被更改。
.material Material 材质

方法：

公用方法可以参考 基类Ojbect3D

Methods params return marks
.raycast() raycaster : Raycaster
intersects : Array		 null 获取光线投射与当前点的交点
.clone() null Points 返回当前点对象

  render(){
    return(
      <div ref="canvasBox" className="canvas-box"></div>
    )
  }

  componentDidMount(){
    this.run()
    document.addEventListener('mousemove', this.mousemoveHandle.bind(this), false)
  }

  initCamera(){
    this.camera = new THREE.PerspectiveCamera(55, window.innerWidth/window.innerHeight, 2, 2000)
    this.camera.position.z = 1000
    this.camera.lookAt( this.scene.position );
  }

  initScene(){
    this.scene = new THREE.Scene()
    this.scene.fog = new THREE.FogExp2(0x000000, 0.0008)
  }

  initRenderer(){
    this.renderer = new THREE.WebGLRenderer()
    this.renderer.setSize(window.innerWidth, window.innerHeight)
    this.refs.canvasBox.appendChild(this.renderer.domElement)
  }


  initModel(){
    let geometry = new THREE.BufferGeometry()
    let vertices = []
    let texture = new THREE.TextureLoader().load(this.ballIcon)
    for (let i=0; i<1000; i++){
      let x = 2500 * Math.random() - 1000
      let y = 2500 * Math.random() - 1000
      let z = 2500 * Math.random() - 1000
      vertices.push( x, y, z )
    }

    // BufferAttribute 只接受类型化的数组作为参数
    geometry.addAttribute( 'position', new THREE.BufferAttribute( new Float32Array(vertices), 3 ) )
    this.material = new THREE.PointsMaterial( { size: 30, sizeAttenuation: false, map: texture, alphaTest: 0.5, transparent: true } )
    this.material.color.setRGB(Math.random(), Math.random(), Math.random())
    let point = new THREE.Points(geometry, this.material)
    this.scene.add(point)
  }

  webGLrender(){
    this.timemap += 1
    if (this.timemap > Number.MAX_VALUE/2){ this.timemap = 0 }
    if (this.timemap % 77 === 0){
      this.material.color.setRGB(Math.random(), Math.random(), Math.random())
    }

    this.camera.position.x += ( this.mouse.x - this.camera.position.x ) * 0.05;
    this.camera.position.y += ( -this.mouse.y - this.camera.position.y ) * 0.05;
    this.renderer.render(this.scene, this.camera)
  }

  animate(){
    this.webGLrender()
    requestAnimationFrame(this.animate.bind(this))
  }

  run(){
    this.initScene()
    this.initCamera()
    this.initRenderer()
    this.initModel()
    this.animate()
  }

  mousemoveHandle(e){
    this.mouse.x = e.clientX - window.innerWidth/2
    this.mouse.y = e.clientY - window.innerHeight/2
  }

}

export default WebglPointsBillboards
