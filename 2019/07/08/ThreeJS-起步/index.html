什么是WebGL

WebGL（全写Web Graphics Library）是一种3D绘图标准或说协议。它允准把Javascript和OpenGL ES 2.0结合在一起，通过增加OpenGL ES 2.0 的一个Javascript绑定，WebGL可以为HTML5的Canvas提供硬件3D加速渲染。
事实上就是使用webgl作用为壳子来进行Open GL接口的调用。因此核心代码譬如着色器, 材质， 光线等仍旧需要使用Open GL的语法进行操作。
说白了就是使用Javascript来写3D

检查你的浏览器是否支持WebGL： 打开 https://get.webgl.org/，如果看到下图所示，则表明浏览器是支持webGL的

什么是THREE.js

THREE.js是一种使用Javascript语言编写的一款运行在浏览器的3D引擎。它封装了底层的图形接口，对外提供API供开发人员使用。 它地位相当于JQuery之于Javascript， 开发人员无需深入掌握高深的图形学知识，仅仅使用其API即能创建3D场景。除了WebGL之外，Three.js还提供了基于Canvas、SVG标签的渲染器。

其功能如下

  1. 根据开发人员的需求方便快捷地创建3D场景
  2. 为物体的渲染提供多种类型的纹理和材质
  3. 自带阴影计算功能，可实现逼真的阴影效果
  4. 引擎中带有多种着色器，可实现多种逼真的效果

Three.js 是Github上的一个开源项目，地址 https://github.com/mrdoob/three.js

下载

首先，我们需要在Github下载Three.js的代码。
在下载过来的代码中可以看到three.js和three.min.js两个文件，前者是没有经过代码压缩的，因此适用于调试阶段；后者是经过代码压缩的，调试起来会不太方便，但文件较小，适用于最终的发布版。
或者使用npm包管理工具

> npm i three

引用

我们需要在HTML中引用 three.js 文件

<script type="text/javascript" src="three.js"></script>

然后就能通过全局变量THREE进行相应开发工作了

或者使用 import 语法 

import * as THREE from 'three'

THREE.js 基本组件

一个典型的THREE项目，至少需要包含渲染器（Renderer）、场景（Scene）、照相机（Camera）三大组件, 以及在场景中创建的物体，几何体(Geometry)，甚至还有 材质(material)以及光线(light)
