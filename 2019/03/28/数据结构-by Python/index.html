用python对数据结构进行解释

链表

定义

链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。
一个简单的单链表节点由两部分组成：一是数据部分，另一个是指针部分。数据部分用于存储数据，指针部分用于记录链上的下一个结点。
单链表中有两个特殊结点： 头结点以及尾结点。 头结点是记录着整个链表的基础地址，通过头结点遍历整个链表。 尾结点是整个链表的终点，它的指针指向 Null (在python中是None)。

优点

由于链表是无序的，仅通过指针相连。因此在执行 插入/删除 等操作时，只需要进行对应指针的更改。其时间复杂度为 O(1)

缺点

同样由于链表是无序的，随机访问第K个元素的时候，需要从头开始检索，一直到目标元素为止。其时间复杂度为 O(n)

与数组的区别

  1. 链表与数组(顺序线性表)都属线性表的一种。
  2. 数组在内存中是连续序排列，通过改变数组大小改变的。而且链表可以是无序的，节点间通过指针连接。
  3. 数组的对内存空间的要求高，必需有足够的连接内存空间。链表对内存空间的利用率较高，基本上不会浪费内存
  4. 数组的大小固定，如内容超过数组空间，需重新定义数组。链表无固定大小，易于扩展
  5. 与链表相反，数组是连续的，很容易定位到对应内存地址的数据。因此查找数组内元素的时间复杂度为O(1)，而对数组的插入以及删除操作都会导致之后数据的内存地址偏移，因此其时间复杂度为O(n)

简单单链表的实现 

class Node():
    def __init__(self, data, next = None):
        self.__data = data
        self.__next = next

    @property
    def data(self):
        return self.__data

    @data.setter
    def data(self, data):
        self.__data = data

    @property
    def next(self):
        return self.__next

    @next.setter
    def next(self, next):
        self.__next = next

class SinglelinkList():
    def __init__(self):
        self.__head = None

    def get_value(self, value):
        # 通过数据值来获取对应节点
        node = self.__head
        if node != None and node.data != value:
            node = node.next
        else:
            return node

    def find_index_node(self, index):
        # 查询第index个节点
        node = self.__head
        pos = 0
        while node != None and pos != index:
            node = node.next
            pos += 1
        return node

    def find_pro_node(self, node):
        # 查询对应节点的前一节点
        if node == None or node == self.__head:
            return
        # 找到指定节点的前一节点
        pro_node = self.__head
        while pro_node.next != node:
            pro_node = pro_node.next
        return pro_node

    def add_to_head(self, value):
        # 在头部插入节点
        node = Node(value)
        node.next = self.__head
        self.__head = node

    def add_to_last(self, value):
        # 在尾部插入节点
        node = self.__head
        while node != None and node.next != None:
            node = node.next
        node.next = Node(value)

    def add_to_after(self, node, value):
        # 在对应节点后插入节点
        if node == None:
            return
        new_node = Node(value)
        new_node.next = node.next
        node.next = new_node

    def add_to_before(self, node, value):
        # 在对应节点前插入节点
        if node == None or self.__head == None:
            return
        if node == self.__head:
            self.add_to_head(value)
            return

        # 找到指定节点的前一节点
        pro_node = self.find_pro_node(node)
        edit_node = Node(value)
        pro_node.next = edit_node
        edit_node.next = node

    def del_node(self, node):
        # 删除指定节点
        if self.__head == None:
            return
        if node == self.__head:
            self.__head = node.next
            return
        pro_node = self.find_pro_node(node)
        pro_node.next = node.next

    def reverse(self):
        # 翻转节点
        if self.__head == None:
            return None
        if self.__head.next == None:
            return self.__head
        p = self.__head.next
        while p.next != None:
            q = p.next
            p.next = q.next
            q.next = self.__head.next
            self.__head.next = q

        p.next = self.__head
        self.__head = p.next.next
        p.next.next = None
        return self.__head

    def getMiddleNode(self):
        # 找出中间节点
        # 快慢指针法 同时使用两种时间，一个逐个指针遍历， 一个跨两指针遍历，当快的指针遍历完链表时， 慢的指针刚好走到链表的一半。其时间复杂度为 O(n)， 空间复杂度为 O(1)
        fast = self.__head
        slow = self.__head
        if not fast.next:
            return self.__head
        if not fast.next.next:
            return self.__head.next

        while fast and fast.next != None:
            slow = slow.next
            fast = fast.next.next
        return slow

    def print_list(self):
        # 打印链表节点
        node = self.__head
        while node != None:
            print(node.data)
            node = node.next
