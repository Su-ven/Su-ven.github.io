在vue/react中关于静态资源的别名引用

发表于 2020-07-01 | 分类于 前端
对于静态资源别名的引用方法，react与vue大同小异

对于别名的使用方法

通常而已言，别名路径的使用场景只有三种，html、js、css 调用

在JS中，可以使用 import或者require 语句进行调用

如果需要在方法或者变量内使用，可以直接写require引用

注：在require中使用模板字符串变量，或者直接使用变量，编辑都会报警告

可以通过字符串+变量的方法进行编写，就不会再出现警告了

css中使用，需要加入“~”，并且不要写成字符串

html中使用，可以加入‘~’也可以不加入‘~’
