在VUE中重置data数据

发表于 2020-07-13 | 分类于 前端
当data中的数据太多，不可能每一项都重新去赋值清除，因此需要一个方法进行快捷重置

需求

对 VUE 中的data数据进行重置

现状

项目中有对data数据进行重置的需求，检查旧代码，发现是这样写

带来问题

代码冗长，阅读困难。data每新增一个属性，clear方法需要同步维护一次，维护成本高

解决

利用VUE中 this.$data 以及 this.$option.data() 特性来进行data的重置

$data

见于VUE官方文档的解析 https://cn.vuejs.org/v2/api/#data
